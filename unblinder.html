UNBLINDER

Alpha testing - currently only for experimentation

Unblinder is a tool to re-identify assignments after marking.

For this to work, upload a zip file containing student assignments along with the marking_table.tsv and checksums.md5 files and Unblinder will rename the files to their original name.
